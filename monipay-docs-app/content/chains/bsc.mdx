# Monipay on BSC

Send USDT (BEP-20) on BNB Smart Chain with zero gas using Monipay.

## Network Details

- **Chain ID:** `56`
- **Token:** USDT
- **Decimals:** 18
- **Relayer Model:** EIP-712 Meta-Transactions
- **Explorer:** [BscScan](https://bscscan.com)

## Implementation Notes

Monipay on BSC uses a recompiled version of the `MonipayRouter` to handle the 18-decimal nature of BEP-20 USDT. The architecture mirrors the Base implementation, providing a consistent experience across EVM networks.
