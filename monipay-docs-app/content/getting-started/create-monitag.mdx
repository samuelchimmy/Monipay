# Create your MoniTag

Your MoniTag is your universal payment identity.

## Setup Process

1. Visit [monipay.xyz](https://monipay.xyz).
2. Choose a unique username between 3 and 20 characters.
3. Use only lowercase letters, numbers, and underscores.
4. Set a 6-digit PIN to secure your local wallet.

## Reserved Usernames

Some usernames are reserved for system use or brand protection. If your desired tag is taken, try a variation. See the [reserved list](/monitag/reserved-usernames) for details.

## Casing and Storage

MoniTags are stored in lowercase. While the UI might display them as you typed, the underlying resolution always uses the lowercase version.
