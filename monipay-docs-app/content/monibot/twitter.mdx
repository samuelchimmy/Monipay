# MoniBot for Twitter / X

MoniBot enables seamless multilevel financial intent execution (transfers, escrow, giveaways, and tipping) on Twitter/X via mention commands.

## Commands

- `@Monipay send $5 to @username` , Tip a user in a reply or new tweet.
- `@Monipay drop $100` , Initialize a giveaway.

## Constraints

Due to Twitter's platform policies:
- MoniBot uses an 18-character hash limit for command tracking.
- Links are sometimes restricted; MoniBot uses verified mention replies to guide users to their claims.
