# Person-to-person payments

Monipay's core primitive is the P2P payment. Send value to any MoniTag in seconds.

## Key Features

- **No Gas:** Monipay sponsors the transaction fees.
- **Username Resolution:** No need to handle long addresses.
- **Cross-Chain:** Send USDC on Base or USDT on BSC from the same interface.
- **Instant Finality:** Payments settle on-chain as soon as the block is confirmed.

## User Flow

1. Enter recipient `@MoniTag`.
2. Input amount.
3. Sign with 6-digit PIN.
4. Transaction is broadcast and confirmed.
