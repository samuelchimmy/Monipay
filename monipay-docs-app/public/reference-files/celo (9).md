# MoniPay on Celo (MiniPay)

MoniPay runs as a **mini-app inside Opera's MiniPay** wallet on Android, bringing MoniTag identity to Celo's cUSD-first user base.

- **Token:** cUSD
- **Decimals:** 18
- **Network:** Celo Mainnet
- Gasless via the MonipayRouterCelo contract
- MoniBot and merchant tools work identically inside MiniPay

This is MoniPay's path into emerging markets where MiniPay already has users and on-ramps.
