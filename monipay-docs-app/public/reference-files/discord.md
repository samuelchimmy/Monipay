# Progressive Web App

MoniPay is a fully-featured PWA. Install it on iOS via Safari → Share → Add to Home Screen, or on Android via Chrome → Install App.

- **Service worker:** network-first cache strategy with offline fallback
- **Smart install prompt:** dismissible, with 7-day suppression via `localStorage`
- **Push notifications** for received payments (Android)
- **Deep links** open the PWA when installed
