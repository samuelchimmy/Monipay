# MoniBot for Telegram

DM `@MoniPayBot` on Telegram or add it to any group. Linking uses Telegram's `return_to` deep-link callback to your MoniPay account.

Commands mirror Discord: `send`, `airdrop`, `grant`, `balance`, with cross-chain keywords (`on base`, `on bsc`, `on tempo`).

## Linking requirements

Only the **sender** needs to link Telegram to their MoniTag‚ĄĘ. Recipients without a MoniPay account receive a **MagicPay** claim link they can redeem securely at any time.
