# Base contracts

- **MoniPayRouter.sol** — verified on BaseScan
- **MoniBotRouter.sol** — verified on BaseScan
- **Builder code:** ERC-8021 attribution `bc_qt9yxo1d` is appended to every MoniPay-routed Base transaction as a hex suffix.

The contracts implement EIP-712 typed-data verification, atomic 99/1 split, and nonce-based replay protection.
